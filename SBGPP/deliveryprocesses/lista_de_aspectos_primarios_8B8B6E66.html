Work Product Descriptor (Artifact): Lista de Aspectos Primários
A Lista de Aspectos Primários é a lista principal de todos os aspectos desejados no jogo.

Main Description

Quando o projeto de um jogo é iniciado não há nenhum esforço abrangente, que demanda muito tempo, para escrever todos os aspectos do jogo. Normalmente, tudo o que é óbvio consta do projeto, o que, quase sempre, é mais que suficiente para o primeiro Estágio. A partir daí, a Lista de Aspectos Primários deverá crescer e mudar na medida em que se desenvolve o jogo.

Durante a reunião Reunião de Planejamento do Estágio, o Gerente do Jogo prioriza os aspectos da Lista de Aspectos Primários e os descreve aos times Audiovisual e de Mecânicas e Jogabilidade. Os times, então, determinam quais aspectos eles conseguirão concluir no próximo Estágio e passa estes aspectos do Lista de Aspectos Primários para o Guia de Aspectos do Estágio. Conceitualmente, os times iniciam do topo da lista priorizada da Lista de Aspectos Primários e desenham uma linha logo abaixo dos aspectos que os times consideraram ser o último possível de se concluir durante o Estágio.

Aspectos da Lista de Aspectos Primários podem ser aspectos técnicos ("O jogo deve trabalhar com grande quandade de elementos sem perda de desempenho em dispositivos móveis") ou mais focados no usuário ("O jogador pode interagir com todos os elementos do cenário").
