Task: Priorizando os Aspectos Primários
Os aspectos da Lista de Aspectos Primários são priorizados pelo Gerente do Jogo previamente ao Plano de Lançamentos e ao Planejamento do Estágio.
Purpose

Para garantir que o time construa o jogo correto.

Main Description

A Lista de Aspectos Primários é uma lista priorizada de trabalhos a serem seguidas pelo time. É continuamente, atualizada pelo Gerente do Jogo que se baseia no feedback vindo do usuário final (à medida em que são expostos a novos incrementos) e, também, em resposta às evoluções técnicas e cenários de negócios

É importante que o time abasteça o Gerente do Jogo com tamanhos estimados para os itens de Lista de Aspectos Primários para, permitir ao Gerente do Jogo balancear as necessidades do negócio com as estimativas de custo para implementação.
